Python library tutorials
========================

Executed Jupyter notebooks introducing the Galacticus Python library
interface — the physics engine of Galacticus exposed to Python through the
``libgalacticus`` shared library.  Each notebook is committed with its
executed output (kept up to date by continuous integration against every
build of the library), so the results below are real.  To run them
yourself, see the `tutorials README
<https://github.com/galacticusorg/galacticus/tree/master/tutorials>`_ for
how to obtain the library.
